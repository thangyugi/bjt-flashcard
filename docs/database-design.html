BJT Flashcard — Database Design

Tài liệu thiết kế cơ sở dữ liệu cho ứng dụng học từ vựng tiếng Nhật business (BJT).
Sử dụng Supabase (PostgreSQL). Phiên bản hiện tại: v1.0

📊 Table Schema

vocabulary_sources — Nguồn tài liệu từ vựng (BJT, JLPT...)
ColumnTypeKeyNullableDescription
idUUIDPKNOT NULLPrimary key, auto-generated
nameVARCHAR(100)NOT NULLTên nguồn (e.g. "BJT 1000 Từ Vựng")
descriptionTEXTNULLMô tả chi tiết nguồn
source_typeVARCHAR(20)IDXNOT NULLEnum: 'BJT' | 'JLPT' | 'OTHER'
levelVARCHAR(10)NULLe.g. "N1", "N2", "BJT-J1"
created_atTIMESTAMPTZNOT NULLDEFAULT NOW()
vocabulary_groups — Nhóm từ vựng theo chủ đề
ColumnTypeKeyNullableDescription
idVARCHAR(50)PKNOT NULLe.g. "bjt-g01", "n1-g01"
source_idUUIDFKNOT NULL→ vocabulary_sources.id
nameVARCHAR(100)NOT NULLTên nhóm tiếng Việt (e.g. "Họp & Thảo luận")
name_jaVARCHAR(100)NULLTên nhóm tiếng Nhật (e.g. "会議・話し合い")
descriptionTEXTNULLMô tả ngắn về nhóm
theme_colorVARCHAR(20)NOT NULLHex color (e.g. "#3B82F6")
iconVARCHAR(10)NULLEmoji icon (e.g. "🗣️")
sort_orderINTEGERIDXNOT NULLThứ tự hiển thị
created_atTIMESTAMPTZNOT NULLDEFAULT NOW()
vocabulary_items — Từ vựng đơn lẻ với đầy đủ metadata
ColumnTypeKeyNullableDescription
idVARCHAR(50)PKNOT NULLe.g. "bjt-001", "n1-015"
group_idVARCHAR(50)FKIDXNOT NULL→ vocabulary_groups.id
kanjiVARCHAR(100)NOT NULLTừ chính (kanji hoặc katakana)
hiraganaVARCHAR(200)NULLFurigana (cách đọc)
romajiVARCHAR(200)NULLRomaji phiên âm
katakanaVARCHAR(200)NULLDùng cho từ ngoại lai
meaning_vnTEXTNOT NULLNghĩa tiếng Việt (bắt buộc)
meaning_enTEXTNULLNghĩa tiếng Anh
meaning_jaTEXTNULLGiải thích bằng tiếng Nhật
part_of_speechVARCHAR(30)NULLnoun/verb/adjective/expression/katakana...
jlpt_levelVARCHAR(5)IDXNULLN1/N2/N3/N4/N5
bjt_relevanceBOOLEANIDXNOT NULLDEFAULT false. True = từ BJT
example_sentenceTEXTNULLCâu ví dụ tiếng Nhật
example_translationTEXTNULLDịch câu ví dụ
business_contextTEXTNULLGhi chú ngữ cảnh business Nhật
usage_notesTEXTNULLLưu ý cách dùng, phân biệt từ
related_wordsTEXT[]NULLArray IDs của từ liên quan
synonymsTEXT[]NULLTừ đồng nghĩa
antonymsTEXT[]NULLTừ trái nghĩa
sort_orderINTEGERNOT NULLThứ tự trong nhóm
created_atTIMESTAMPTZNOT NULLDEFAULT NOW()
updated_atTIMESTAMPTZNOT NULLDEFAULT NOW(), auto-update on change
flashcard_progress — Theo dõi tiến độ học từng từ (Phase 2: thêm user_id)
ColumnTypeKeyNullableDescription
idUUIDPKNOT NULLPrimary key
vocabulary_item_idVARCHAR(50)FKIDXNOT NULL→ vocabulary_items.id
user_idUUIDIDXNULL→ auth.users.id (Phase 2)
statusVARCHAR(20)NOT NULL'new' | 'learning' | 'review' | 'mastered'
review_countINTEGERNOT NULLDEFAULT 0. Số lần đã ôn
next_review_atTIMESTAMPTZNULLThời điểm ôn lại tiếp theo (SRS)
last_reviewed_atTIMESTAMPTZNULLLần ôn gần nhất
created_atTIMESTAMPTZNOT NULLDEFAULT NOW()
updated_atTIMESTAMPTZNOT NULLDEFAULT NOW()
study_sessions — Phiên học theo nhóm (Phase 2)
ColumnTypeKeyNullableDescription
idUUIDPKNOT NULLPrimary key
group_idVARCHAR(50)FKNOT NULL→ vocabulary_groups.id
user_idUUIDIDXNULL→ auth.users.id (Phase 2)
started_atTIMESTAMPTZNOT NULLDEFAULT NOW()
ended_atTIMESTAMPTZNULLNULL nếu session chưa kết thúc
cards_studiedINTEGERNOT NULLDEFAULT 0
cards_correctINTEGERNOT NULLDEFAULT 0

⚙️ Migration Guide — Phase 1 → Phase 2

⚠️
Phase 2 Prerequisites
Trước khi migrate, cần: (1) Tạo project Supabase, (2) Setup environment variables, (3) Install @supabase/supabase-js, (4) Thay hooks trong useVocabulary.ts.
🔧 Phase 2 Migration Steps
# 1. Tạo Supabase project tại supabase.com
# 2. Thêm environment variables vào .env.local
NEXT_PUBLIC_SUPABASE_URL=https://xxxxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJxxx...

# 3. Install Supabase client
npm install @supabase/supabase-js @supabase/ssr

# 4. Tạo file src/lib/supabase.ts
# 5. Chạy migration SQL lên Supabase SQL Editor
# 6. Chạy seed SQL files theo thứ tự: 00 → 01 → 02
# 7. Cập nhật useVocabulary.ts để gọi Supabase thay dummy data
✅
Tip: Backward Compatible
Hooks useAllVocabularyGroups, useVocabularyGroup... được thiết kế để chỉ cần thay đổi phần fetcher bên trong, không cần sửa components. Chuyển đổi Supabase sẽ rất dễ dàng!
